# LLM Recommender System

## Overview
An LLM-powered recommender system designed to generate personalized recommendations using embeddings, semantic similarity, and language models.

## Key Features
- Embedding-based recommendation
- Semantic search / similarity matching
- LLM-driven ranking or explanation layer

## Tech Stack
- Python
- LLMs (OpenAI / HuggingFace / Local models)
- Vector similarity (FAISS / Chroma / Pinecone – update as used)
- Pandas, NumPy
- Jupyter Notebooks

## Project Structure
src/        → core pipeline, models, utilities  
notebooks/  → EDA, experiments, prompt testing  
data/       → local datasets (gitignored)  
models/     → saved models (gitignored)  
outputs/    → results, evaluations (gitignored)
